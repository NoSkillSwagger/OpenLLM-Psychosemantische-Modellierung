# OpenLLM-Psychosemantische-Modellierung

## Thema
Bedeutungsräume und typische Assoziationen in großen Sprachmodellen (LLMs) im Vergleich zu Menschen.

## Fragestellung
Zeigen LLMs ähnliche semantische Verwandtschaften, Assoziationen und Bedeutungscluster wie Menschen?

## Beispiele

- **Freie Assoziation:** „Sag das erste Wort zu `Hund`.“
- **Ähnlichkeitsbewertung:**  
  Klassifikation und Ähnlichkeiten von Wörtern wie *Katze*, *Leine*, *Tier*, *Spielzeug*.
- **Sortieraufgabe:**  
  Sortieren Sie die Wörter in Gruppen, von denen Sie denken, dass sie thematisch oder inhaltlich zusammengehören.

## Ziel des Projekts
Erstellung eines quantitativen Vergleichs zwischen LLM-assoziierten Bedeutungsrelationen und menschlichen Assoziationsmustern; Identifikation von Gemeinsamkeiten und Abweichungen.

## Verfahrensweise

- Erstellung Fragekatalog für verschiedene Aufgaben.
- Setup Ollama mit verschiedenen Open LLMs.
- Datenerhebung mit LLMs.
- Datenerhebung mit Menschen.
- Analyse und Vergleich.

## Arbeitsaufteilung

- **Gemeinsam**
  - Erstellung der Fragekataloge
  - Vergleich

- **Lukas**
  - Datenerhebung LLMs
  - Analyse

- **Peer**
  - Datenerhebung Menschen
  - Analyse
